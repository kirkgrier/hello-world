This page links to the various staging servers created by SiteGround as we demonstrate (DEMO site), develop (TEST site) changes to:

https://standrewspcusa.org

The DEMO site is https://dptdemo.standrewspcusa.org/

The TEST site is https://dpttest.standrewspcusa.org/

The staging servers are automatically named by SiteGround, and the DEMO and TEST mappings need to be manually updated. So if a DEMO or TEST link is not working, you can check the direct "stagingXX" links below. If a server is up, you will connect to it without any errors, or getting a "default vhost" screen. Finally, depending on where we are in development, DEMO and TEST servers may not be set up, or they may be the same server. Ideally we will have a notice on the home page of the site you connect to telling you if it is DEMO or TEST.

Please report any issues to kgg at gcssba.com. - Thanks, Kirk
